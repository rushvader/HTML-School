My HTML Revision

HTML stands for Hypertext Markup Language. It can use another file called a stylesheet to change the look of the webpage. HTML uses "tags" to change the style of text it uses. For example, < p> is used for paragraphs

To make text go bold, use the < b> tag. You can even make them bold italic. And you can underline them with < u>

  • < body> - Used for the page's body
  • < h1> - Makes a large header
  • < i> - Makes the text it wraps around italic
  • < b> - Makes the text it wraps around bold
  • < li> - Makes an unordered list
  • < p> - Makes a paragraph
  • < p align=center> - Makes a center text
  • < br> causes a line brake
Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
